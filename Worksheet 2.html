Chart challenge

1. Scatterplot

Is there a relationship between carats and price? Make a scatterplot that helps us see the relationship. Make sure you think about which variable goes on which axis.

ggplot(data = diamonds, aes(x = carat, y = price))+
  geom_point()

2. Histogram

What is the distribution of carats in the dataset? Make a histogram showing this distribution.

ggplot(data = diamonds, aes (x = carat))+
  geom_histogram(bins = 30, color = "white")

3. Boxplots

What is the distribution of prices, split by cut? Notice that cut is a categorical variable, so we can use a boxplot for this visual.

ggplot(data = diamonds, aes(x = cut, y = price))+
  geom_boxplot()

4. Barplots

How many diamonds are there in each cut category? Make a barplot to show this.

ggplot(data = diamonds, aes(x = cut))+
  geom_bar()

5. Get fancy.

Choose one of your existing charts and modify a graphical element, such as color of your encodings.

ggplot(data = diamonds, aes(x = cut))+
  geom_bar(color = 'black', fill = 'steelblue')
